# 🐶 TinDog

A modern, responsive landing page for a fictional dog dating platform built with **React**, **Vite**, and **Bootstrap 5**.

This project demonstrates responsive frontend development, reusable React components, clean UI design, and component-based architecture. It was created as a portfolio project to practice building modern landing pages using React.

---

## 🚀 Live Demo

👉 https://bassil88.github.io/TinDog-Project/

## ✨ Features

* Responsive landing page
* Component-based React architecture
* Modern Bootstrap 5 layout
* Pricing section
* Testimonials section
* Feature highlights
* Call-to-action section
* Footer with social links

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Bassil88/TinDog-Project.git
```

Navigate into the project

```bash
cd TinDog-Project
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
